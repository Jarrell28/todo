### Setup

Run ``` npm i ``` to install the following dependencies:

 - react
 - react-dom
 - react-scripts
 - react-bootstrap
 - react-jsonschema-form
 - react-router-dom
 - bootstrap
 - uuid
 - node-sass
 
 **Initialize Application**

``` npm start ```
